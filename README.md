# Spaghettos
Simple Operating System created for didattical purposes of understading how all works under the hood. The goal of the project was also to learn better usage of C/C++ (with slight bias toward the latter), that are the standard languages for this type of Softwares. 

The current project follows [this project structure](https://www.open-std.org/jtc1/sc22/wg21/docs/papers/2018/p1204r0.html). 

You can find all the [Theory](./theory/Index.md) about all the project in the linked files.
