# Spaghettos

Simple Operating System created for didattical purposes of understading how all works under the hood. The goal of the project was also to learn better usage of system languages (C/C++/Rust), that are the standard languages for this type of Softwares.

You can find all the [Theory](./theory/Index.md) about all the project in the linked files.

## Kernels

You can find in this repository a really simple kernel, and try to run it with `make run`. It will show a simple image that you can't interact with. For all the kernels, I decided to develop them in external repositories, so you can find them here:

- [Rust Kernel](https://github.com/MatteoPossamai/spaghettos_kernel_rs)
